# Ferramentas do Hacker Ético

O arsenal de um hacker ético é composto por diversas ferramentas especializadas em diferentes fases de um teste de invasão.

## Principais Categorias

### 1. Reconhecimento e Scanning
- **Nmap**: O padrão ouro para descoberta de rede e auditoria de segurança.
- **Dnsenum / Dig**: Ferramentas para enumeração de DNS.

### 2. Análise de Vulnerabilidades
- **Nessus / OpenVAS**: Scanners de vulnerabilidades automatizados.
- **Nikto**: Focado em servidores web.

### 3. Ataques Web
- **Burp Suite**: Proxy de interceptação essencial para testes web.
- **OWASP ZAP**: Alternativa open-source ao Burp.
- **SQLMap**: Automatização de injeção SQL.

### 4. Exploração
- **Metasploit Framework**: Plataforma completa para desenvolvimento e execução de exploits.
- **Searchsploit**: Busca offline no exploit-db.

### 5. Quebra de Senhas
- **John the Ripper**: Cracker de senhas versátil.
- **Hashcat**: Cracker de senhas acelerado por GPU.

## Ética e Legalidade
O uso dessas ferramentas deve ser restrito a ambientes controlados ou sistemas para os quais você tem autorização explícita por escrito. O acesso não autorizado é crime.
